# NexusApp

A Web3 NFT minting application that allows users to connect their wallet and mint NFTs through a simple, user-friendly interface.

## Overview
NexusApp is a frontend-focused Web3 project built to demonstrate NFT minting flows, wallet interaction, and clean UI design for blockchain applications.

The goal of this project is to showcase how NFT products can be made accessible and intuitive for everyday users.

## Features
- Wallet connection
- NFT minting flow
- Transaction status feedback
- Clean and responsive UI
- Mobile-friendly design

## Tech Stack
- React
- Tailwind CSS
- Wagmi
- RainbowKit
- Viem
- Solidity (NFT smart contract)

## How It Works
1. User connects their wallet
2. Selects or previews the NFT
3. Initiates mint transaction
4. NFT is minted and confirmed on-chain
